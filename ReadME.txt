Chrome Extension Name: Page Summerizer

Features:
- Open short summary on context popup on link hover.
- Context popup contain linked page title, one image related to the page and paragraph taken rom the page.
- Expension PopUp contains the extension setting page.
	- User can set following things:
		- Show/Hide Page Title of the linked page.
		- Show/Hide Image related to linked page.
		- Show/Hide short paragraph from the linked page.

External CSS/Javascript used:
- jQuery
- Bootstrap CSS
